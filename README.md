# NLUBOTTRAINER

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

## Project Overview
NLUBOTTRAINER is a project designed to build and train NLU models for chatbot applications. It includes dataset management, model training, prediction, and evaluation functionalities.

## Features
- Upload and manage datasets
- Train machine learning and NLU models
- Make predictions using trained models
- Evaluate model performance
- Interactive dashboard and workspace

## Installation

1. Clone the repository:
```bash
git clone https://github.com/renusrijeyapandiyan/NLUBOTTRAINER.git
Navigate to the project directory:

cd NLUBOTTRAINER
Install dependencies:

npm install
# or for Python backend
pip install -r drizzle/python-backend/requirements.txt
Run the project:

# For frontend
npm run dev

# For Python backend (if using Rasa/ML services)
cd drizzle/python-backend
./start.sh
Usage
Access the dashboard at http://localhost:3000

Upload datasets via the workspace

Train models using the ML/ NLU module

Make predictions and evaluate models
